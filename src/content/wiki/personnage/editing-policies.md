## Categories

All pages and images must be categorised to provide an organised structure that enables future readers and contributors to find other pages. Categories should be added to the end of each page.

Sub-pages should only have Main-pages' name.

### Images

Files should be named properly so that they are easily searchable.

All Images should be from official sources.

Since the Novel has a Manhua we prefer to illustrations from there.

### Vandalism

Vandalism is not tolerated under any circumstances.

Users have put in their effort into building up pages, so try to not mess things up.

If you are found to be vandalising pages, you will be blocked from editing for a certain amount of time.

If you are a frequent offender, the time you are blocked can be increased.
